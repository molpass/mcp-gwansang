# 관상(觀相) 해석

손발(측정)과 지식(해석)을 나눈다.
- **손발 = `mcp_gwansang_analyze_face` MCP.** 사진 → 결정론 측정 수치(삼정(三停) 비율·오관(五官) 비율·대칭·자세). 해석은 0.
- **지식 = 이 스킬.** 그 수치를 관상 의미로 옮긴다. **수치 밖은 만들지 않는다.**

## 철칙 — 측정 없이는 관상 없다

사진이 보인다고 눈으로 직접 "코가 크니 재물이…" 하지 않는다. 도구가 준 숫자만 근거다.
숫자 없는 관상은 재현성도 전거도 없는 환각이다. 이건 양보 대상이 아니다.

| 유혹(실제로 나왔던 것) | 현실 |
|---|---|
| "사진이 보이니 그냥 보고 말하면 되지" | 비전 추측 관상 = 숫자 0·재현 불가·전거 0 = 차단할 환각 모드. |
| 도구 실패 → "비전 설명 기반 수동 분석으로 회귀" | **금지.** 실패는 재시도·재촬영 안내로 갚는다. 추측으로 메우지 않는다. |
| "상정(上停)이 작으니 초년운이 약하다" | 발제선 보정 실패(`fallback`)면 上停은 중립. 보정 여부 확인 없이 단정 금지. |
| "도구엔 없지만 귀(耳)·기색(氣色)을 보면…" | 측정 안 한 부위는 "이 도구로는 측정 안 됨"이라 말하고 끝. 지어내지 않는다. |

## 호출 흐름 — 1회 호출 → 분기 → 단일 해석

`mcp_gwansang_analyze_face(image=<사진 경로>)`를 **딱 한 번** 호출한다(게이트웨이가 사진 경로를 넘긴다).
한 번에 완전한 피처를 돌려주므로 **추가 도구 호출은 0.** 부족하면 재호출이 아니라 재촬영을 안내한다.
**overlay는 요청하지 않는다 — `include_overlay`는 기본 False로 둔다.** overlay(검증용 PNG)는 사람 눈 검수용 자산이지 해석 입력이 아니다. 대용량 base64가 결과를 잘라 파싱을 깨뜨린다(오관 빈 헤더·raw 노출의 원인).

## 분기 게이트 — 해석 전 반드시 통과

1. `is_person: false` → 관상 **0줄**. `reason`을 반영해 일상어로 재촬영을 안내(얼굴이 또렷이·정면으로 나오게).
2. 도구 에러 / `features` 없음 → 관상 **0줄**, 안내. **비전 추측으로 때우기 절대 금지.**
3. `confidence: "low"` → 단정 톤을 낮추고 `notes` 사유에 걸린 피처를 잠근다. 예: 측면 의심(`yaw_balance` 낮음·`notes`에 측면) → 대칭·좌우 비율 해석 보류. 흐림/작은 얼굴 → 전체 강도 하향 + 재촬영 권유.
4. `features.samjeong_meta.hairline_method == "fallback"` → **상정(上停)·초년운 해석 보류(중립).** `notes`에 "발제선 추정 실패"가 있다. 나머지 피처(중정·하정·오관·얼굴형)는 정상 해석한다.
5. `features.ogwan`이 비었거나 기대 키(삼정(三停)·오관(五官)·얼굴형)가 없음 → **결과 잘림·파싱 실패 신호.** 관상 0줄, 재시도·재촬영 안내. 빈 항목을 추측으로 채우지 않는다(도구 실패 규율의 잘림판).

## 6규율 (측정의 한계를 해석으로 메우지 않기)

- **① 전거 인용 의무.** 모든 해석 문장은 근거가 된 `features` 키·값에 묶인다. 못 묶으면 말하지 않는다.
- **② 해석-금지 게이트.** 위 분기의 비-정상 출력에는 관상 0줄.
- **③ 신뢰도 → 단정 강도.** `confidence`·`notes`·`pose`로 단정의 세기를 조절한다(낮으면 "경향" 톤).
- **④ 계통편향 인지.** 上停·높이 정규화 피처(`face_shape_ratio`·`nose_length_ratio`·`brow_eye_span_ratio`)는 발제선 보정 반영값이다. `hairline_method` 미확인 단정 금지.
- **⑤ 자기-핑거프린트 · 내부값 비노출.** `samjeong_meta.upper_raw`·`geometry_px.face_height_raw`·overlay는 **내부 검증/핑거프린트용** — 사용자 출력에 **노출 금지.** 상정(上停) 해석은 항상 보정값 `samjeong.upper`로 하고 `hairline_method`로 게이팅한다(`fallback`이면 중립). 요약이 측정과 모순되면 신뢰하지 말고 플래그(진짜 호출이면 사람 사진에서 raw가 보정값과 다르게 나온다).
- **⑥ 단일 호출 강제.** 1회로 완전 피처. 부족하면 재촬영 안내(재호출 아님).

## 수치 → 해석 매핑표

전통 마의상법(麻衣相法)·삼정오관(三停五官)의 통상 대응이다. 중립값(아래) 대비 **편차 방향(큼/작음)을 경향으로** 말하고, 항상 근거 수치를 함께 댄다. 절대 중립값은 잠정이다(특히 여성·연령 캘리브레이션은 후속 — 단정 톤을 키우지 않는다).

### 삼정(三停) — 중립 0.33 / 0.33 / 0.33 (균등)
전거: 마의상법 삼정론(三停論) — "상정주조년(上停主早年), 중정주중년(中停主中年), 하정주만년(下停主晚年)".
- **상정(上停)** `samjeong.upper` — 초년운(15~30)·지력·이상·윗사람 덕. 발달(>0.36)=총명·이상·초년 발복 / 빈약(<0.28)=초년 실리·고생.
  ⚠️ **`hairline_method=="segment"`일 때만 단정.** `"fallback"`이면 上停은 중립(초년운 단정 금지).
- **중정(中停)** `samjeong.middle` — 중년운(31~50)·의지·실행력·자립·사회활동. 발달=추진·성취 / 빈약=의존·추진력 약.
- **하정(下停)** `samjeong.lower` — 말년운(51~)·생활력·애정·포용·자식·아랫사람. 발달=말년 안정·정 / 빈약=말년 외로움 경향.
- 상·중·하 중 도드라지게 큰 곳이 그 시기의 강세. (上停은 보정 여부에 따라 비교에서 제외할 수 있음.)

### 오관(五官) — 삼정오안(三庭五眼) 기준 편차
전거: 오관 직책 — 눈=감찰관(監察官), 눈썹=보수관(保壽官), 코=심변관(審辨官), 입=출납관(出納官).
- **눈(目)** `ogwan.eye_length_ratio` (중립 ≈0.20) — 감찰관. 큼=감수성·표현·정·총명 / 작음=신중·내성·집중.
- **양안 간격** `ogwan.interocular_ratio` (중립 ≈0.20, 측정중앙 ~0.25) — 넓음=포용·느긋·시야 / 좁음=예민·집중·성급.
- **눈썹(眉)** `ogwan.brow_eye_span_ratio` = 눈썹~눈 거리 (중립 ≈0.10~0.12) — 보수관·전택(田宅). 넓음=여유·관대 / 좁음=급함·예민·집중.
- **코(鼻)** `ogwan.nose_width_ratio`(콧방울)·`ogwan.nose_length_ratio`(길이) — 심변관·재백(財帛)·자아·중년. 콧방울 넓음=재물·생활력·실리 / 좁음=섬세·소비 신중. 길음=자존·고집·자기 기준 / 짧음=융통·적응.
- **입(口)** `ogwan.mouth_width_ratio` (중립 ≈0.35~0.40) — 출납관·식록(食祿). 큼=적극·리더십·사교·식복 / 작음=신중·말 아낌·내실.

### 얼굴형·대칭·자세
- **얼굴형** `face_shape_ratio` (세로/가로, 중립 계란형 1.3~1.4) — >1.45 갸름·길음(사고형·민감·신경, 목형(木形) 경향) / <1.25 둥글·네모(실행·체력·친화, 토수형(土水形) 경향).
- **대칭** `symmetry` (0~1) — 높음=균형·안정·일관 / 낮음(<0.9)=기복·다면성. ⚠️ `pose.yaw_balance` 낮거나 `roll_deg` 크면 촬영 탓일 수 있어 **보류**.
- **자세** `pose` — 해석 대상이 **아니라** 신뢰 게이트. 비대칭·비율 왜곡이 측면·기울임 탓인지 가른다.

## 톤 — 2모드

- **기본 = 일상어 풀이.** 한자·전문용어 최소(예: "상정"이 아니라 "이마 부위"). 쉽고 따뜻하게. 입발린 소리는 안 쓰되, 부정도 회피 말고 근거와 함께 부드럽게.
- **"근거"·"왜"·"전거"를 물으면 = 전문용어 모드.** 전통 상법 용어를 **한글(한자)** 로 쓰고 수치 전거를 노출한다.
  예: "중정(中停) 비율 0.30, 고전 균등 0.33 대비 −0.03 → 중년 추진력은 평이한 편. (전거: `features.samjeong.middle`)"
- 전거는 두 모드 모두 **내부적으로 항상 추적**한다. 표면에 드러내느냐만 모드에 따라 다르다.

## 범위 잠금 — 측정한 것만 말한다

측정된 5묶음(삼정·오관·얼굴형·대칭·자세)만 해석한다.
**귀(耳)·기색(氣色)·점·주름·십이궁(十二宮) 등 미측정 부위는 언급 금지** — MCP v0 범위 밖이라 말하면 그게 환각이다.

## 한자 표기
통한자 단독 금지. 반드시 한글(한자) — 상정(上停), 오관(五官), 재백(財帛). 처음 한 번 병기 후 본문은 한글로 풀어도 된다.

## 면책
구조 측정에 기반한 해석이며 운명론이 아니다. 참고·재미로 본다.

## 검증 (진짜 도구 호출인지)
- 사람 사진이면 `samjeong_meta.upper_raw`가 보정된 `samjeong.upper`와 다르게 나오는 게 정상(발제선 보정의 서명). 둘이 똑같으면 `hairline_method`가 `fallback`인지 확인.
- `confidence`·`notes`가 출력 형태와 맞물려 일관적인지 본다(예: 측면 사진인데 `notes`에 측면 의심이 없으면 의심).
